# Rock Paper Scissors

A simple implementation of the classic "Rock, Paper, Scissors" game.

## Table of Contents

- [Overview](#overview)
- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)

## Overview

This project provides an interactive game of Rock, Paper, Scissors, where a user can play against the computer. It can be used as a fun project to learn about basic programming concepts and user interaction.

## Features

- Play Rock, Paper, Scissors against the computer.
- Handles user input and validates choices.
- Keeps track of the score (optional—edit if implemented).
- Simple and easy-to-understand code.
- Can be easily extended for new features.

## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Firezer27/rock-paper-scissors.git
   cd rock-paper-scissors
   ```

## Usage

Run the game:

```sh
run index.html and open console
```

Follow the on-screen instructions to select Rock, Paper, or Scissors and play against the computer.
